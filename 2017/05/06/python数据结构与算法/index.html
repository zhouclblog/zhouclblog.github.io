发表于 2017-05-06

一、线性表

根据线性表的实际存储方式，分为两种实现模型：

  • 顺序表，将元素顺序地存放在一块连续的存储区里，元素间的顺序关系由它们的存储顺序自然表示。
  • 链表，将元素存放在通过链接构造起来的一系列存储块中。

    1、顺序表

    python中的顺序表是采用分离式存储的，若想更换数据区，只需将表信息区中的数据区链接地址更新即可，而该顺序表对象不变。列表就是简单的顺序表。

    2、链表

    链表（Linked list）是一种常见的基础数据结构，是一种线性表，但是不像顺序表一样连续存储数据，而是在每一个节点（数据存储单元）里存放下一个节点的位置信息（即地址）。

  • 单项链表的实现：

    class SingleNote(object):
    # 创建一个节点

    def __init__(self, item):
      # item用来存储数据元素
      self.item = item
      #next用来存储下一个节点的标识
      self.next = None

class SingleLink(object):
    # 实现一个单链表的结构，以及封装相应的操作方法

    def __init__(self):
        # 初始化链表的起始位置
        self._head = None

    def is_empty(self):
        # 用来判断链表是否为空
        return self._head == None

    def length(self):
        # 用来求取链表的长度
        cur = self._head  # 用一个标记来记录操作节点时的位置
        count = 0
        while cur ！= None：
            count += 1
            cur = cur.next
        return count

    def travel(self):
        # 调用这个方法，就可以将链表中的所有元素都显示出来
        cur = self._head

        while cur != None:
            print(cur.item)
            cur = cur.next
        print("")

    def add(self, item):
        # 从头部添加元素
        node = SingleNote(item)
        node.next = self._head
        self._head = node

    def append(self, item):
        # 从尾部添加元素
        node = SingleNote(item)
        if self.is_empty():
            self._head = node
            return
        cur = self._head
        while cur.next != None:
            cur = cur.next

        cur.next = node

    def insert(self, pos, item):
        # 从指定位置插入数据
        if pos <= 0:
            self.add(item)

        elif pos > (self.length()-1):
            self.append(item)

        else；
            node = SingleNote(item)

            count = 0

            # 保存插入前该位置对象
            pre = self._head
            while count < (pos-1):
                count += 1 
                pre = pre.next

            # 循环退出后找到要插入的位置的前一个对象
            node.next = pre.next
            pre.next = node

    def remove(self, item):
        # 删除指定数据
        cur = self._head
        pre = None

        while cur != None:
            if cur.item == item:
                if not pre:
                    pre = self._head
                else:
                    pre.next = cur.next

                break
            else:
                pre = cur
                cur = cur.next

    def search(self, item):
        cur = self._head

        while cur != None:
            if cur.item == item:
                return True

            cur = cur.next

        return False
